Java Yolculuğu (Java Journey) 🚀
Bu depoda, Bursa Teknik Üniversitesi Bilgisayar Mühendisliği öğrencisi olarak Java öğrenme sürecimde geliştirdiğim temel projeleri, algoritma çalışmalarımı ve kod örneklerimi topluyorum.

📁 Projelerim
1. Vücut Kitle İndeksi (VKİ) Hesaplayıcı

• Kullanıcıdan boy uzunluğu ve kilo bilgilerini alarak VKİ değerini hesaplar.

• Scanner sınıfı kullanılarak dinamik girdi alma pratiği yapılmıştır.

• Temel matematiksel operatörler ve değişken yönetimi uygulanmıştır.


2. Basit Hesap Makinesi

• switch-case yapısını kullanarak dört işlem (toplama, çıkarma, çarpma, bölme) gerçekleştirir.

• Mantıksal kontroller ile (sıfıra bölünme hatası gibi) daha güvenli bir kod yapısı oluşturulmuştur.

• Kullanıcı etkileşimli terminal menüsü tasarımı yapılmıştır.


🛠️ Kullanılan Teknolojiler ve Araçlar
•Dil: Java

•IDE: Eclipse

•Sürüm Kontrolü: Git & GitHub
